# 豆瓣 Top250 电影数据分析

使用 Python 爬虫获取豆瓣电影 Top250 数据，并进行评分分布、类型频次统计等可视化分析。

## 数据来源

- 数据爬取自 [豆瓣电影 Top250](https://movie.douban.com/top250)
- 爬虫工具：`requests` + `BeautifulSoup`
- 数据存储：`douban_top250.csv`（包含电影名、评分、类型）

## 分析内容

## 关键结论

- **“剧情”** 是豆瓣高分电影最核心的类型，超过 74% 的电影包含剧情元素（186/250）。
- 爱情、喜剧、冒险、奇幻、犯罪也是常见类型。
- 纪录片、运动、灾难等类型数量极少，说明观众偏好有深度故事或娱乐性的电影。
